# Getting Started
Install the dependencies and run the project
```
npm install
npm start
```

Head over to https://vitejs.dev/ to learn more about configuring vite

## About this project

This project is a simple digital business card built using HTML and CSS. It was created as part of my learning journey to strengthen my frontend development skills, particularly in layout design, styling, and responsive design.

The project was developed by following a frontend course from Scrimba and then implemented independently to reinforce concepts.
